培養魅力的3個關鍵 - ⟪魅力學⟫ 讀後心得&重點整理

魅力不是天生的，而是可以透過後天鍛鍊和培養出來的

作者 / 陳文學醫師     2023/10/26

前言

會想讀這本書，主要是因為它被列為⟪職場大人學⟫這堂課的補充教材，出於對「魅力」的好奇，以及想要好好經營職場形象，就特別買了這本⟪魅力學：無往不利的自我經營術⟫來讀，

發現到原來魅力不是天生的，而是可以透過後天鍛鍊和培養出來的，關鍵就在於臨在感、影響力、親和力這三種基本能力

魅力是什麼？

書本的一開始，作者就很直接的說明什麼是魅力，她將魅力拆解成三個基本元素，分別是臨在感、影響力、親和力

1.臨在感

臨在感的英文是「presence」，直白的翻譯就是「存在感」。這個你應該非常有感，有魅力的人，他們的存在感是相當強烈的，當有魅力的人出現在一個空間，就算不發一語，你還是會注意到他的存在

而這背後到底是什麼在運作，造成這樣存在感的差異呢？

作者提到就是你個人對當下的察覺，是否有做到全神貫注不分心。舉例來說，你可以注意到臨在感低的人，常常眼神渙散、心不在焉、一心多用，對於當下的事情沒有全心全意的投入

這讓我想到過去有一位存在感很低的前同事，在開會時總是在角落用電腦做自己的事，真的在會議中存在感就低到不行

而臨在感強的人，則會專注於當下，跟他對談時，你會覺得他是全心全意在關注你，回應你

書中提到像微軟的比爾蓋茲、特斯拉的馬斯克等等都是有強大臨在感的人，他們不一定很外向很健談，但是能給人一種他是全心全意關注你的感覺，所以也能顯得非常有魅力

2.影響力

影響力講的粗俗一點，就是讓別人覺得你是不是一個「咖」的能力。有影響力的人，會讓人感覺到他可能有權、有勢或是擁有支配資源的能力

這部分就很社會性，主要是透過肢體語言、外表穿著打扮、頭銜、別人對你的反應來影響

最明顯的例子就是各國的國家領袖，你可以發現他們的穿著打扮、肢體語言都很類似，能給人一種權威的感覺

這讓我想到老高與小茉的youtube頻道，確實當老高說完話以後，小茉人馬上給予認同的眼神和贊同的話，就會給觀眾一種老高非常有影響力的感覺

3.親和力

第三個魅力的元素是親和力，這是一種讓別人判斷你是朋友還是敵人的能力。遇到親和力強的人，你就會很自然的想和他做朋友，和他站在同一陣線。相反的，親和力低的人，你的大腦就會自動把他判斷成敵人

書中舉的例子我覺得比較遙遠一點，像是德雷莎修女和達賴喇嘛，他們會給人一種很有愛心、很慈祥和善的氣場

作者提到這三種能力當中，臨在感是最為重要的能力，它是魅力的地基，就算你有很強的影響力和親和力，但沒有足夠的臨在感的話，是沒有辦法帶給別人有魅力的感覺，因爲別人會覺得你心不在焉，沒有專注在他身上

這讓我想到自己在看門診的時候，很常會一邊問診一邊打病歷，雖然這樣作業流程上會比較快，但其實會給病人一種心不在焉的感覺，比較好的做法應該是先看著病患，仔細聽完主述和問診完以後再開始打病歷，當然這樣作業上會拖比較久，但我覺得至少在頭一分鐘應該要做到這樣，才能給人好感

魅力其實來自於強大的內心

接下來就來到本書最有趣的地方了，到底要如何培養自己的魅力，讓自己成為有魅力的人呢？ 還是說魅力是天生的，沒有辦法透過訓練來培養？

這點可以放心，魅力是真的可以訓練的，作者的本業就是專門幫企業的主管們訓練出魅力，讓他們可以更順利的帶領團隊

不過訓練的方法可能跟你想的不一樣，第一步不是學怎麼穿著打扮，而是先學會調整自己的內心狀態

意識能控制的肢體動作相當有限

你可會覺得，魅力不就是來自那外表bling bling的樣子嗎？

作者在書中是這樣解釋的，魅力很大一部分是來自你的肢體動作，你可能想說，那我不就是學一些看起來很有魅力的肢體動作就好了嗎？

但很實際的是，你的意識所能掌控的肢體動作其實相當有限，因為你很難一心多用，很難一邊控制自己臉上的表情，一邊又注意自己的腳是擺成什麼樣的姿勢，然後一邊又要控制自己講話的語調

而根本的解決方法，就是讓你的潛意識自然而然的讓肢體呈現出有魅力的動作。而要達到這種狀態，重點就是要訓練你的內心，讓你的內心強大起來

內在能力對魅力的影響

作者做過一個有趣的調查，她訪談了一些基層工程師、執行長還有一些業界的魅力人士，請他們對自己的專業能力、外在能力、內在能力做自我評價，結果發現了有趣的現象，基層工程師普遍認為自己有高度的專業能力、普通的外在能力和較差的內在能力，而魅力人士則認爲自己的專業能力不怎麼樣，但是有強大的外在能力和內在能力

所以你可以很明顯的認知到，魅力的根源是強大的內心，有了強大的內心，你就能靠潛意識讓身體自然而然的呈現出有魅力的肢體語言

這讓我想到之前在拍形象照的時候，不管攝影師怎麼樣說：「再笑開一點」，我還是一直都笑得很不自然，後來攝影師說：「想一下你這輩子最開心的事情」，我馬上就笑得很燦爛很自然


如何培養魅力?

暸解了魅力其實來自強大的內心以後，接下來就來分享一些書中提到訓練魅力的練習，而這些練習大致上可以分成這幾個階段

  1. 訓練臨在感：強化你對當下的專注力ㄋ，讓你可以不分心，全心全意的投入
  2. 排除心理障礙：有很多負面情緒像是焦慮、恐懼等等會影響你的魅力，這部分的練習可以讓你消除這些負面情緒
  3. 設定內心狀態：透過一些方法，讓自己的心理狀態可以設定在比較正向、高昂的狀態
  4. 調整外在：最後透過外表穿著、打扮、肢體語言、說話語調讓魅力外顯

1.訓練臨在感

訓練臨在感的效果，就在當你在和對方講話時，不會給人一種心不在焉的感覺，可以讓你的思緒專注在當下

以下書中提到訓練臨在感的方法：

臨在感

  1. 找個安靜的地方，讓你能閉上雙眼一分鐘(站著或坐著都行)

  2. 將計時器設定為一分鐘，閉上雙眼，試著把注意放在以下三件事情：周遭的聲音、你的呼吸、或是你腳趾頭的感覺。

  3. 周遭的聲音：聆聽四周環境有什麼動靜，把你的耳朵想像成衛星天線，被動且客觀地接受聲音

  4. 你的呼吸：不但要專注在呼吸上，還要體會空氣在鼻腔進出時的感覺。只專注在呼氣和吸氣，試著感受吸、吐之間的變化，把呼吸當成是你全神貫注的事情

  5. 你的腳指頭：全神貫注在你腳趾頭的感覺，這麼做能讓你的心思貫穿全身，有助於你專注當下的知覺

其實練習方式和冥想還蠻接近的，都是訓練你對當下的專注力，讓自己的心思不要飄來飄去，這是一個簡單但相當有效的訓練

2.排除心理障礙

書中排除心理障礙的練習非常多，這邊分享一個自己也很常困惱的東西，那就是面對不確定的事情時所產生的焦慮感，

作者用一種叫做「責任轉移法」的方式，可以化解這樣的焦慮，簡單地說，他是用想像的方式，把責任轉交給全能的神、上帝or宇宙whatever，用這種方法可以避免大腦反覆去想各種可能的後果，而產生緊張、焦慮感。

以下是詳細的步驟：

責任轉移法

  1. 舒服地坐著或是躺下，放輕鬆，把眼睛閉上

  2. 做兩三次深呼吸。吸氣時，想像自己將純淨的空氣引導到頭頂。呼氣時，讓這股氣習快速流竄全身，洗淨所有的憂慮與煩惱

  3. 選擇某個仁慈的神祈，上帝、命運之神或宇宙都行，是個人信仰而定

  4. 想像把肩上扛著的憂慮一一卸下，眼前的會議、對話、這一天，轉移到你選定的神祈肩上。現在，你所有的憂慮全轉交祂們

  5. 觀想自己肩上的所有重責全數卸下，現在的你不必再對任何事物的結果負責，感受一下前後的差異。每件事都已獲得託管，你現在可以好好坐下，放鬆享受這美好的過程

3.設定內心狀態

第三部分，作者分享了一些可以協助你進入正向心態的練習，其中的「自我慈悲」練習是我之前從來沒聽過的東西，但是照著練完幾次，真的讓我比較能夠接納自己，承受挫折的能力也跟著提高

慈悲

  1. 舒適地坐著，閉上眼睛，做兩到三次的深呼吸。吸氣時，想像將大量乾淨空氣吸到頭頂。吐氣時，想像那想那團空氣從頭到腳快速沖刷下來，洗淨你所有的煩惱

  2. 回想生活中你曾做過的一件好事，大事小事都可以，它可以是一個善行，也可以是一時的真誠、慷慨或勇氣。將心思暫時專注在這段回憶上

  3. 現在想一個你鍾愛的對象，無生命、有生命，虛構或真實的都可以。例如耶穌、佛祖、德雷莎修女或達賴喇嘛。對象可以是人或寵物，甚至填充動物玩偶也可以

  4. 在心中描繪出這個對象的樣貌，想像他們的親切、仁慈、慈悲心。從他們的眼裡與臉上觀察這些特質，感覺他們對你散發出一陣陣溫暖，包圍著你。

  5. 透過他們親切仁慈的眼神來看待自己，感受他們原諒你內心批判的一切錯誤。如今，你獲得了全然、百分之百的寬恕，你就是一張全新的白紙

  6. 感受他們全心全意的包容你：接受你本來的樣貌，無論是現在或是成長過程，不管你完美與否，他們全盤接受

自我慈悲和自尊不太一樣，自我慈悲是一種反脆弱的概念，像是彈力球，允許自己受挫，防止過度的自我責備，而自尊則是一種剛強的概念，極力維持一種很強大的形象，但就像玻璃球，掉到地上就碎了，不允許自己有任何失誤或挫敗，有的話就直接崩潰

4.調整外在

外在可以調整的空間很多，主要帶來的影響就是可以增加你的權威型魅力

穿著的部分，如果你想要給人一種權威、強勢的感覺，那你就要穿得比周圍的更加正式一點。但想反的，如果你想要給人一種好親近、是同一夥的感覺，那你就要盡量穿得跟別人類似。

書中提到去新創公司面試應徵的例子，有人想要應徵一家公司，就特別跑到那家公司的門口觀察他們的員工，看看他們都是穿什麼樣的衣服，然後在面試當天盡量穿得類似，就會給人一種同一夥的感覺

而聲音的部分，如果要給人沈穩的感覺，作者提到講話就要緩慢而規律、音調壓低，然後適時地停頓，展現自己的自信

肢體部分的話，減少一些不必要的小動作，例如說玩頭髮、玩筆、重新排列桌上的東西等等，都可以增加自己的權威度

不要小看外在的影響力，很多時候，外在的肢動作會反過來影響內在心裡

總結

以上就是簡單紀錄和分享這本⟪魅力學：無往不利的自我經營術⟫的一些心得，這本書不是一本讀過去就好的書，裡面有很多練習需要自己去練過以後才會有功效，我自己也還在實踐當中，也慢慢改進一些小動作，讓自己的肢體可以更和諧，如果對經營自己職場形象有興趣的人，非常推薦這本書
