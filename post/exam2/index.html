[上課進修] 從事勞工健康服務醫師訓練二階課程 上課心得

課程內容不會太難，最難的大概就是報名課程了，真的非常搶手

作者 / 陳文學醫師     2021/10/03

這個月到台中上了110年度「從事勞工健康服務醫師訓練」課程，學到醫師在從事臨場服務所需具備的基本知識 醫師到事業單位進行臨場服務的法源依據，主要是依照勞工健康保護規則第3條所提到的：「事業單位之同一工作場所，勞工總人數在三百人以上或從事特別危害健康作業之勞工總人數在一百人以上者，應視該場所之規模及性質，分別依附表二與附表三所定之人力配置及臨場服務頻率，僱用或特約從事勞工健康服務之醫師及僱用從事勞工健康服務之護理人員，辦理臨場健康服務。」 而醫師在臨場服務可以做的項目有：

  1. 勞工體格（健康）檢查結果之分析與評估、健康管理及資料保存
  2. 協助雇主選配勞工
  3. 健檢異常者之追蹤管理及健康指導
  4. 高風險勞工之評估及個案管理
  5. 健康教育、衛生指導、身心健康保護、健康促進等措施之策劃及實施ㄋ
  6. 工作相關傷病之預防、健康諮詢與急救及緊急處置。

職場健康促進

關於職場的健康促進(Health promotion)，依據耶魯大學公共衛生教授的定義是「一門預防疾病、延長壽命、增進健康與效率的科學與藝術，經由有組織的社區力量，從事環境衛生、傳染病管制、個人衛生教育」。根據WHO渥太華健康促進憲章，健康促進是一種增能賦權(Empowerment)，經由這個過程使人們能夠控制其健康決定因子，並因而改善健康。和職業安全衛生不同，職場健康促進是一種由下而上的方式，讓員工參與和充權。實務上的作法是去調查員工想做什麼，例如員工表示沒有時間和場域運動，可以透過美化廠區的樓梯，讓員工願意透過走樓梯來達到運動的效果，重點在於提供勞工足夠的健康資源，讓員工可以有能力去選擇。也有企業會開發APP，讓員工可以紀錄運動的時數，有些員工餐廳也會強化營養和熱量的標示，或是透過餐盒的分隔設計，讓員工可以自由選擇適合的食物，並且透過餐卡可以協助追蹤員工所攝取的熱量。

工廠訪視與職業衛生

關於工廠訪視與職業衛生，主要法源是依據勞工健康保護規則第12條，雇主應使醫護人員、勞工健康服務相關人員配合職業安全衛生、人力資源管理及相關部門人員訪視現場，辨識與評估工作場所環境、作業及組織內部影響勞工身心健康之危害因子，並提出改善措施之建議。主要危害辨識可以從化學性、物理性、生物性、人因工程危害這些方面著手。化學性危害可以先看工廠使用之化學品清單，並檢視其物質安全資料表(MSDS)，暸解其毒性，並觀察作業流程中可能進入人體的途徑。物理性危害又有噪音、高溫、震動、游離性輻射/非游離性輻射等等。人因工程危害可以先以傷病現況調查以及「肌肉骨症狀調查表」(Nordic Musculoskeletal Questionnaire,NMQ)，做初步篩檢，並以KIM評估表做作業流程評估。生物性危害需注意現場環境對有害物質有無標示和管制。

配工

關於配工，主法源依據是職業安全衛生法第21條，健康檢查發現勞工有異常情形者，應由醫護人員提供其健康指導；其經醫師健康評估結果，不能適應原有工作者，應參採醫師之建議，變更其作業場所、更換工作或縮短工作時間，並採取健康管理措施。以及勞工健康保護規則第19條，屬於第四級管理者，經醫師評估現場仍有工作危害因子之暴露者，應採取危害控制及相關管理措。於勞工健康保護規則中的附表11，有說明考量不適合從事作業之疾病，但並非絕對禁止，需視實際狀況調整。主要方向為依照健康風險之嚴重性、以及健康風險發生頻做評估，如果健康風險高，且風險發生頻率也高，就需要調整工作內容或方式，暫停暴露，其他的話則是提供足夠的資訊供員工參考。配工也可參考Fitness for work:The Medical Aspects書籍。

復工

關於復工，根據加拿大醫學會的共識：「強烈建議臨床醫師儘早讓病患重返工作崗位，長時間工作請假，不利於個人心理、生理、社會層面之健康，臨床醫師應建議病患生病或受傷後，儘早重返職務與工作」。復工的評估主要由風險(Risk)、體能(Capicity)、耐受性(Tolerance)這三方面去著手。風險指的是員工做可以做，但因為疾病可能會勞工自己、同事、社會造成傷害，所以應該禁止(restriction)。體能則是勞工當前能力不足，應給與limitation 和處方訓練。耐受性則是勞工承受某種工作或活動的能力，是勞工主觀的感受，為心理身理學概念，無法客觀測量或驗證。關於復工，也可以參考AMA Guide to the Evaluation of Work Abilitu and Return to Work，對於各種受傷的停工休養天數都有詳細建議。臨場醫師和臨床醫師不同之處，在於臨場醫師應正向強調患者還有的工作能力，而非失去的能力，讓員工先從事較輕的工作，逐步恢復原職位，漸進式復工。

後記

課程內容不會太難，最難的大概就是報名課程了，真的非常搶手，一分鐘就額滿的那種，建議有想要上的人，一有空檔就一定要先去上一上
