[開箱] LAMY Studio 鋼珠筆 - 打造專業的職場書寫形象

作者 / 陳文學醫師     2023/12/03

以建立職場專業形象來說，台幣2000多的LAMY Studio系列的鋼珠筆會是個很好的選擇，而且它使用的筆芯是M63鋼珠筆筆芯，與一般常見Pilot、Pentel、 MUJI等平價筆芯在長度上只有一點小差別，做一些小加工後就可以替代使用，節省更換筆芯的費用

前言

這個月聽到大人學的Bryan分享派克的第五元素鋼珠筆，突然讓我意識到，我在工作上其實需要一支看起來比較「正式」的筆

在住院醫師的時候，我使用的都是無印良品的按壓式膠墨筆，主要是為了應付病房混亂充滿突發狀況的工作環境，需要這種不見也不心疼，可以亂丟亂擺、替代性高的筆

但升上主治醫師以後工作環境就變了，除了居家醫療以外，大部分時間都是靜態坐著，筆其實不太會亂丟，而且像臨廠服務時，還真的有人會care你用的東西，有一次就有人跟我抱怨我的水瓶太舊了，看起來不順眼，讓想當然書寫的筆一定也是有人會注意，所以就下定決心買一隻好筆，自己看得順眼、也讓病人看得順眼，並且要讓人感覺寫出來的字是有重量的，左挑右選，最後選擇了Lamy Studio 鋼珠筆

關於LAMY Studio系列

LAMY是來自德國的鋼筆公司，其最知名的筆款就是最初階的LAMY Safari，在誠品和各大文具店都可以看到它的身影，價格落在1000元上下，而再進階一點的筆款，就是LAMY aion、LAMY Studio、LAMY 2000這三款筆型，價錢從2000～3000多都有，三種筆款的筆身材質和造型各有不同，因為我在實際使用上也不會到大量書寫，所以就以外型做為主要考量

LAMY studio是在2005年由瑞士工業設計師Hannes Wettstein與LAMY合作推出的系列，使用包浩斯風格來設計此筆款，其中最吸引我的就是筆蓋蓋上時，整枝筆看起來是一體成型的，不會特別有「筆蓋」的感覺，相當俐落，再來是筆夾的部分採用立體的螺旋槳造型，給人有個性不呆板的感覺

這次買的黑森林(Black Forest)顏色是2021年的特別款，和一般版的霧面黑色不一樣，黑森林雖然也是黑色，但是是亮面的材質，再燈光照射下會帶有一點亮灰色，很像汽車烤漆的感覺

開箱

外盒

LAMY studio 的筆盒其實還蠻簡單樸素的，就是灰色的紙盒

裡頭透過紙卡和彈力帶固定住筆

紙卡下方沒有額外的說明書，包裝很簡單

這個盒子自用的話我是覺得ok，但如果是要送禮用的話，真的稍嫌單薄了一點，想當初買Sailor的鋼筆時，價位差不多的筆款，Sailor可是給了一個紮實的收納盒呢

本體

接下來就是LAMY studio的本體啦，亮面銀黑色的筆身相當的洗練

頭尾都是亮面不鏽鋼的材質

最具代表性的螺旋槳式筆夾，帶有些許飛行航空感

筆蓋上印有小小的LAMY logo

將筆拆卸開來，發現結構還蠻簡單的，使用的是LAMY M63鋼珠筆筆芯M63的粗細體感大概是0.7mm，還蠻粗的

改裝

接下來進到改裝的階段，要將原本原廠的M63鋼珠筆芯改成無印良品的0.5mm中性筆筆芯

首先可以看到，M63筆芯和無印良品的中性筆筆芯其實很類似，在筆頭的不銹鋼部位、筆芯寬度等等都近乎相同

只有在最尾端有小小的長度差別

接著只要把這一小段長度補齊就可以了，mobile 01上有人是用螺絲、橡皮筋或橡皮擦，我是用同樣無印良品用過的筆芯，剪一小段下來補齊長度

放進筆桿後鎖上，長度剛剛好

換上MUJI的筆芯以後，寫起來的觸感比想象中的好很多，沒想到相同筆芯用不同的筆身為差那麼多，給家人試寫，都說沒想到居然是無印良品的筆芯

總結

最後來個新舊筆大對比

一比較才發現，原來無印良品的筆比LAMY studio還要長，不過LAMY的份量感和質感真的是好上太多了

細節是堆疊出來的，希望透過這支筆，可以讓病人在診間時更能感受到服務是認真的、專業的！
